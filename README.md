# סימולציה דגלי דוד — ניתוח מונים + אגירה

דאשבורד אינטראקטיבי לניתוח קריאות מונה (15 דק') של חברת החשמל, שכבת עלות תעו"ז, וסימולציית אגירה.
לקוח: **ד.ג.מ. דוד סחר בינ"ל** · חוזה 000341580196.

## חי בענן
**https://nadavsemo-a11y.github.io/dagli-david-sim/** (GitHub Pages · push=deploy · ללא אבטחה).

## עיצוב
כל הצבעים/הפונטים נמשכים מ-`theme.css` — **מקור עיצובי יחיד** הנגזר ממערכת enSights
(primary `#124948`, darkest `#0a2828`, accent `#33ae9a`, info `#1eb7ff`, danger `#ed1c24`, warning `#f27212`; פונט Arial/Helvetica).

## אגירה (ארביטראז' תעו"ז)
פיצ'ר סימולציה: טעינה מהרשת בשעות השפל, פריקה בשעות הפסגה לקיזוז יבוא — **ללא יצוא לרשת**.
פרמטרים ניתנים לעריכה: מס' קבינטים, קיבולת/קבינט (261 kWh), הספק AC (125 kW), SOC מקס'/מינ' (95%/20%), נצילות.
גרף טעינה + גרף פריקה + KPI חיסכון (ערך פסגה שקוזז − עלות טעינה בשפל = חיסכון נטו).

## הפעלה מקומית
פתח את **`index.html`** בדפדפן. עובד גם אופליין (Chart.js מקומי).

## קבצים
| קובץ | תפקיד |
|------|-------|
| `index.html` | הממשק |
| `app.js` | לוגיקה: נגזרות, אגרגציה, גרפים, בקרות |
| `tariff.js` | סיווג תעו"ז (עונה/פסגה-שפל) + מחירים (מתח נמוך, בתוקף 01.07.2026) |
| `meter_data.js` | סדרת הזמן המעובדת (נבנה ע"י `build_data.py`) |
| `chart.umd.min.js` | ספריית הגרפים (מקומית) |
| `build_data.py` | עיבוד קובץ ה-CSV הגולמי → `meter_data.js` |

## עדכון נתונים
כשמגיע יצוא CSV חדש מחח"י, עדכן את הנתיב ב-`build_data.py` (משתנה `SRC`) והרץ:
```
python build_data.py
```

## מיפוי המונים (נגזר מהנתונים)
- **ייצור PV** = `PV1 + max(PV2,PV3)`:
  - PV1 = 503-24707415.
  - PV2 = 738-21002689, PV3 = 738-21002691 — **אותה מדידה בדיוק** (מונה בקרה + מסחרי): זהים ב-70% מהסלוטים, בהפרש < 0.1 קוט"ש ב-99.7%, מתאם 0.9985. לכן לוקחים max פר סלוט (לא סכום) — אחרת הייצור מנופח ~פי 2 (שיא צהריים 448kW במקום 249kW האמיתיים, שתואם AC < 300kW).
- **דו-כיווני** 737-18000646: עמ' "צריכה"=יבוא מהרשת, עמ' "הזרמה"=יצוא לרשת.
- **מונה צריכה** 737-18000661: קורא בפועל את היבוא מהרשת (זהה כמעט לדו-כיווני).

## גדלים נגזרים
- **צריכה עצמית** = ייצור − יצוא (PV שיוצר ונצרך במקום, מחויב בתעו"ז).
- **סך עומס** = יבוא + צריכה עצמית (סך צריכת המבנה בפועל).

## יכולות הדאשבורד
- **רזולוציות**: 15 דק' / שעה / יום / שבוע / חודש / שנה / רב-שנתי.
- **מדד**: אנרגיה (kWh) / הספק ממוצע (kW) / הספק שיא (kW).
- **סדרות**: סך עומס, יבוא, ייצור PV, יצוא, צריכה עצמית, מונה צריכה (בחירה חופשית).
- **טווח תאריכים**: בורר + פריסטים (7/30/90 יום, שנה, הכל).
- **שכבת תעו"ז**: פילוח האנרגיה/העלות (₪) לפי 6 תעריפים (פסגה/שפל × קיץ/חורף/מעבר), עם/בלי מע"מ, ובחירה על איזו כמות מחשבים (עומס / יבוא / צריכה עצמית).

## הערות
- מחירי התעו"ז העדכניים מיושמים על כל הטווח (לא שוקללו עדכוני תעריף היסטוריים).
- חגים מטופלים כשבתות וערביהם כימי ו', כהגדרת חח"י.
- פערי שעון קיץ/חורף (DST) זניחים (~4 סלוטים בשנה).
